# LMS Backend API

## Features

- JWT Auth (Register & Login)
- Google OAuth ready integration
- MongoDB + Express + TypeScript
- Swagger-ready folder (`src/swagger`)

## Scripts

```bash
pnpm install
pnpm dev
```

## ENV Setup

```env
PORT=5000
MONGO_URI=mongodb://localhost:27017/lms
JWT_SECRET=your_jwt_secret
```
